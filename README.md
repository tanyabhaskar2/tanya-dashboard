# Tanya Dashboard 💸

## Overview
A finance dashboard built using Express and JavaScript to track income and expenses with visual insights.

## Features
- Summary Cards (Balance, Income, Expenses)
- Add & Delete Transactions
- Filters (Income / Expense)
- Charts (Line + Pie)
- Insights (Top Spending Category)
- LocalStorage (data persists)

## Tech Stack
- Node.js
- Express.js
- HTML, CSS, JavaScript
- Chart.js

## How to Run

```bash
node server.js
